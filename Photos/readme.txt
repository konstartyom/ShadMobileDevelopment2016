Приложение "Фото", как следует из названия, позволяет просматривать фотографии. В данный момент фото берутся только из галереи (папка /storage/emulated/0/DCIM/Camera). Соответственно, для работы требуется android.permission.READ_EXTERNAL_STORAGE, и, естественно, папка должна быть непуста. Краткий Feature list:
- Показ фотографий в виде квадратных preview. Количество элементов в ряду задается в настройках
- По клику фотография анимированно увеличивается и занимает весь экран. После чего её можно увеличивать/уменьшать/панорамировать с помощью обычных для мобильных устройств жестов. По кнопке назад/клику на фотографии или вне её она так же анимированно возвращается на место
- Поддержка нескольких вкладок (пока что фотки берутся из одного источника, поэтому содержимое одинаково). Переключаться между вкладками можно жестом
- Меню выезжает по кнопке вверху слева или жесту пальцем
- Поддержка тем. Их две: светлая и тёмная. Primary color: cyan. Выбираются в настройках
- Локализовано пока что только на русский язык (+ английский по умолчанию)
- Настройки тем и числа превьюшек в ряд сохраняются между запусками приложения
- Теоретически должно работать на android, начиная с 4.0.3 (API Level 15). Пока тестировалось только на 5.0.2 (Xiaomi Redmi Note 2)
